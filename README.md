# Todo List Application
This is a simple Todo List application built with HTML, CSS, and JavaScript. It allows users to add, edit, and delete tasks.

## Features
- Add tasks
- Edit tasks
- Delete tasks


## Instructions

# Open `index.html` in your browser to use the application.
